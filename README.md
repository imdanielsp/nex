# Nex Language

A dynamically typed interpreted programming language.

Features:
* First Class Functions
* Type Inference
* Classes

## Building The Project

First, ensure you have `java` and `cmake` installed, then run:

    mkdir build
    cd build
    cmake ..
    make

## Running The Project

To compile a file run:

    build/src/nexc

To run unit tests:

    build/tests/nexc_test
